### 项目简介

- **设计目的：**

  ​		在学习计算机底层原理，如汇编语言、编译原理时，初学者往往学的云里雾里，知其然不是其所以然，所以本项目旨在通过简单的C/C++ 语法构建一个汇编语言模拟器，实现从汇编代码到机器码（汇编）、机器码到汇编代码（反汇编）以及一个模拟的微型计算机去执行由汇编得到的机器码，最后得到程序的运行结果。
